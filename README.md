# Kode Relasi BPS dengan Kemendagri

This repository contains information on obtaining LAU (Local Administrative Units) relation codes conducted by BPS Statistics and the Indonesian Ministry of Home Affairs (Kemendagri). You are welcome to use this script at no cost. However, we kindly ask that you do so thoughtfully and responsibly.

If you would like more detailed information, you can access the website at https://sig.bps.go.id/bridging-kode/index.
